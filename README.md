# Leadership - Evolutionary Game Theory

This repo contains the code used to obtain the results presented in the paper.
The libraries used to run the model with 1 bit, 2 bits, and 4 bits strategies for the single-leader and multi-leader scenarios are contained in the two mechanics folders.
The run_model and run_model__strats scripts run the model and store the cooperation level and the stationary distributions of the strategies and can be modified to choose which model from the libraries to use.
